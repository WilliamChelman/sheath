# Sheath

A companion web app for [DRAW STEEL](https://mcdmproductions.com/), MCDM's tabletop RPG.

**[Try it live](https://williamchelman.github.io/sheath)**

## Features

- **Compendium** - Browse and search game rules, ancestries, classes, abilities, and more
- **Token Generator** - Create custom tokens for your heroes and monsters

## Acknowledgements

Sheath is inspired by and built upon the work of these amazing projects:

- [Forge Steel](https://github.com/andyaiken/forgesteel) by Andy Aiken - A hero builder for Draw Steel

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run start:web-app

# Build for production
npx nx build web-app

# Run tests
npx nx test web-app
```
